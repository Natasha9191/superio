What Is Dashlane?

Dashlane is a password manager, digital wallet, and comprehensive security app. Desktop versions are available for Windows and macOS, mobile apps for Android and iOS, and browser extensions for Chrome, Firefox, Internet Explorer, Safari, and Edge.

The password manager is well-known for its huge selection of features, going far beyond the basics that most password managers provide. Some of its paid packages even include a VPN. Considering all of the above, it's easy to see why Dashlane has built a reputation as one of the best all-rounders out there.

Still, you might be wondering whether Dashlane has spread itself too thin by offering so much. Or maybe, whether the software is user-friendly? Let's find out.

Installation & Usage

It doesn't take long to get Dashlane up and running. Users can download the browser extension without needing to install the desktop app. To do this, simply click on the "Get Dashlane" button on the service's website.

After that, you'll create a master password, but don't worry, the app helps with this too. Once Dashlane has been properly configured, all that's left to do is remember the master password. We recommend choosing something that you can easily remember, but won't be too simple to crack.

Dashlane helps users create a secure master password.

Next, the app guides you through its basic features in a tutorial, showing how to use "Try Autofill" and how to get started with the software. A QR code with a link for the mobile app is also provided to download the service for your smartphone or device.

Dashlane helps users get started with its software.

Should you switch to Dashlane from another password management program, click on My Account > Import Passwords to import data sets (CSV files). We tested this with a CSV file from LastPass and were relatively satisfied with the results. Unfortunately, it wasn't possible to specify which data sets should be imported from the file, and the categories and credit card information we had added in LastPass didn't transfer.

With Dashlane, you can use data sets created with other software.

Dashlane's installation is easy, regardless of your skill level. Should you encounter any problems, the service is there to help every step of the way. We had a few (minor) import issues, but you shouldn't have any difficulties with the software's basic features.

Dashlane reinvents itself

Dashlane seems to be reinventing itself: Support for the desktop version ended in early 2022 and its features have been integrated into the web app.

Below, we'll introduce you to all of Dashlane's components.

Web app

Dashlane's new web app provides access to all basic features in a user-friendly interface. All available data set categories (passwords, secure notes, personal data, payment methods, and IDs) are listed in the menu to the left. There, you'll also find the "Sharing Center", where you can safely share entries, along with "Password Health" and "Dark Web Monitoring".

To the right of the menu bar, all data set entry types in your vault are listed. These can be sorted alphabetically, by category, or by last use. Click on an entry to open a detailed view in which you can edit usernames or passwords. It isn't possible to modify categories, and after querying support, they informed us that this functionality still hasn't been integrated into the new web app.

One of Dashlane's special features is "Password Changer", which you can use to automatically change passwords for specific online services. We'll discuss this in greater detail below, however, this feature is still undergoing beta testing. Other bells and whistles, such as the VPN, haven't made the jump over to the software's browser version.

Access Dashlane's most important features in its web app.

Desktop app

To tap into Dashlane's full potential, you'll need to use its desktop app, which is designed similarly to the web app. To the left of the navigation menu, you'll also find entries (the "receipts" data set type is also included, as is the ability to set an emergency contact). All of your data sets are located in the center of the interface.

The detailed view provides nearly the same options, however, you can assign entries to existing categories and create new ones.

The service's VPN, which is only available to premium subscribers, is accessible via the upper menu bar.

Many Dashlane features are only available in the desktop app, however, the password manager's future is clearly browser-based.

Browser extension

The service's browser extension, which is pinned to your browser's taskbar, serves as the control center for the program's autofill features. You can view all saved entries in the "Vault" tab, with entries for the site that you're presently on listed first. Beyond that, you can adjust autofill settings for each website, or create secure passwords with the password generator. For everything else, you'll be forwarded to the web app.

Data sets for the site you're visiting are displayed first in the browser extension.

Dashlane is in a transitional phase: Its desktop app seems to be on the way out, with the password manager relying on its web console and browser extension. Even though we had no difficulties with these, there were some minor usage issues. We're also disappointed that support for the desktop app will be discontinued.

In our eyes, it seems like Dashlane isn't as well-rounded as it was during our last review. Once the transition from desktop to browser has been completed, and features are concentrated in a single interface, the provider should return to its former glory.

Features

Dashlane has everything you'd expect from a password manager: password management, autofill settings, encrypted notes, and more. But the app offers much more and gives some popular features a dash of extra convenience while including others that its competitors don't have.

Password Changer

One of these is "Password Changer". This handy premium feature allows paying subscribers to change multiple stored passwords with a single click. If you use the same password for every website, Dashlane can help you easily enhance your security with a single click. It's also possible to check how strong your passwords are and change any weak ones.

Unfortunately, the tool doesn't work with every website. From the 300 entries in our sample, only six could be automatically changed (three of these were with the same service). With that said, a new version is undergoing beta testing, and Dashlane constantly adds to the list of supported sites.

Use "Password Changer" to automatically fix unsafe passwords.

Autofill

When it comes to convenience and password managers, it's all about autofill. Like all other services from our sample, Dashlane handles the automatic input of login data and other forms through its browser extension.

Whenever the password manager identifies an input field, the Dashlane icon appears. After clicking on this, you'll be able to select from saved entries and automatically supply a username and password. Dashlane also automatically saves new registration data, lending a hand by offering users the ability to create and save automatically generated passwords.

During testing, Dashlane's autofill feature didn't disappoint: It effortlessly and accurately juggled classic registration forms, addresses, and bank data.

Click on the Dashlane icon to automatically fill in login and password data.

Password Generator

A password manager wouldn't be complete without a password generator. Dashlane's tool is prominently integrated into the browser extension, allowing users to configure and create secure passwords with a single click. On top of that, the service's password generator is easy to find, lightning-fast, and intuitive. The only downside is that it cannot be used to create full passphrases.

Use Dashlane's "Password Generator" to automatically create secure passwords.

Password Health

This feature provides you with status updates for your passwords. You can see how secure each password is by the number of points it has. This score relates to each password's complexity and other factors, such as how often you've used the same password. Change weak passwords as quickly as possible, either manually or with "Password Changer".

With "Password Health" you can monitor the security of your passwords.

All subscribers get "Dark Web Monitoring", which tasks Dashlane's security experts with checking whether your data or passwords have been misused on the Dark Net.

Password Sharing

Individual entries or entire categories can be shared with other Dashlane users in the "Sharing Center". You can also grant these users limited or full access. Those with limited access will be able to use shared data sets, but not modify or share them.

Share passwords and other data sets with friends and family with just a few clicks.

VPN

Starting from its sixth release, Dashlane included a virtual private network (VPN). This offered users an added layer of data security, especially when browsing on public WiFi networks. Compared to dedicated VPNs, Dashlane's was fairly limited and only offered 20 different countries to connect to.

As of the time of writing, the VPN is only accessible via Dashlane's desktop version, with no date set for its return to all of the software's components. For that reason, it remains to be seen whether the VPN will be integrated into the browser extension.

Taken together, Dashlane comes loaded with practical features, offering one of the best packages out there. Unfortunately, some of these features are behind paywalls.

Security

Dashlane saves passwords using AES 256-bit encryption, which is considered to be extremely secure. Just how safe your master password is depends on you. Since there's no way of recovering it, apart from adding an emergency contact, the recovery feature can't be misused to obtain it. This makes Dashlane much more secure than several of its competitors.

Enhance your vault's security by adding two-factor authentication.

Mobile app use poses a potential security risk. In addition to the master password, users can enable fingerprint verification for logging into websites. This is optional though, and using it depends on your security priorities.

Mobile Usage

Dashlane's mobile apps are available for Android and iOS devices. Since data sets are synchronized with your desktop, you'll find the same types and categories on your smartphone as your PC or laptop. The features are identical as well: "Password Health", "Password Changer", "Dark Web Monitoring", and the VPN are all included in the mobile version.

The most important aspect of the mobile app is autofill. The app doesn't have a dedicated browser, however, autofill worked without a hitch in Chrome during testing. When you start typing into an input field, you'll be shown saved usernames and passwords for the website you're on, which you can select using the virtual keyboard. Tap on the Dashlane icon to select from all available accounts.

Dashlane's mobile app offers users access to all important features on the go.

If you become tired of tapping out your master password, you can log in using your fingerprint or a four-number PIN. Facial recognition can also be enabled, so long as your device supports this.

Overall, Dashlane's mobile app is a reliable sidekick that offers more than just the basics, including a number of extras from the desktop version.

Support

There are two ways to contact Dashlane's support team. Monday to Friday, you can use the live chat from 9-5. Each time we tested this, we were helped within less than a minute.

Outside of these times, you can request support through the contact form. For this, we waited around 21 hours for an answer.

Contact Dashlane's support via live chat or email.

In the event that you prefer to solve issues on your own, Dashlane also has a well-curated static support center that includes comprehensive help articles and guides. Here, you'll find a number of articles about the changeover from Dashlane's desktop version to its web app. The help center also features a search function.

Taken together, we felt well looked after by Dashlane, being able to quickly find or receive informative answers to our questions.

Pricing

For individual users, Dashlane offers three packages. Its Free plan is limited to 50 saved passwords and a single device. Of course, you can use the mobile version with the same account, but you won't be able to sync passwords.

To access everything that Dashlane has to offer, you'll need a Premium subscription. In the event that you want multiple Premium accounts, you can book a Family subscription.

Payments can be made either monthly or annually; an annual subscription costs 20% less than a monthly one.

Professional plans (Starter and Business) are also available. The Business package includes SAML-based single sign-on and a free family subscription for each individual license.

Dashlane definitely deserves top billing when it's being ranked. It's good at the basics and has an intuitive dashboard, even if some features that make surfing more secure and comfortable are behind paywalls. All the same, you can rely on Dashlane's autofill for passwords, bank data, and addresses.

These days, Dashlane seems to be changing its approach which has generated a few issues here and there. The desktop version's days are numbered, however, not everything has been integrated into the new web interface. This might make using the software somewhat frustrating, since you'll need to jump between the web app and desktop version, and might forget which features are where.

However, in the grand scheme of things, this is a temporary problem and we're excited about the new web application.

FAQs

What is Dashlane?
⊖

Dashlane is a password manager for Windows, macOS, Android, and iOS. It also serves as a digital wallet and offers a number of additional features including a VPN. As such, Dashlane is widely considered to be one of the best all-rounders in its market segment.

How much does Dashlane cost?
⊖

Dashlane offers both free and premium versions. Users of the free version are limited to 50 passwords and a single device, and cannot sync their desktops with the mobile app. Dashlane Premium costs $4.99 a month and includes advanced security tools. Friends & Family and professional rates are also available.

How safe is Dashlane?
⊖

Dashlane is highly secure since it protects data with AES 256, the highest encryption standard available as of the time of writing. You can also use two-factor authentication to further reduce security risks. However, it's important to create a secure master password that isn't easy for others to guess.

How good is Dashlane?
⊖

Dashlane is one of the best password managers we've reviewed. The company probably offers the strongest package on the market, wrapped in an intuitive user interface and bursting with useful features. The best part? It's available for a very reasonable price.
