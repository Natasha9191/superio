Website Security Check: How Secure Is Your Website?

With the EXPERTE.com Website Security Check you can determine how secure your website is. We'll let you know whether malware was found on your website, whether it appears on blacklists, how secure your SSL certificate is, and whether you are using the right HTTP security headers. A complete list of all security checks our tool performs can be found below, in the next section.

To start the check, simply enter your domain in the field above and confirm by pressing Enter. After a few seconds, you'll receive the following information:

  • Malware & Phishing
    We check whether there's any malware on your site and if it's been classified as a phishing site.

  • Blacklist-Status
    We check whether your domain, IP, mail server, or name server have landed on any known blacklists. This can be an indication that malware or spam are being distributed or sent through your site.

  • SSL Certificate
    We display information on the publisher, hostname, signature algorithm used, and key size. We also check whether the certificate or certificate chain is trustworthy and if your certificates are still valid.

  • SSL Protocols
    We check 6 different SSL protocols and whether they are in use, also providing tips about any which should no longer be used for security reasons. The protocols are SSLv2, SSLv3, TLS 1, TLS 1.1, TLS 1.2, and TLS 1.3.

  • SSL Security Vulnerabilities
    In addition, we check whether your server is vulnerable to any known security threats. Specifically, we test for susceptibility to Heartbleed, POODLE, FREAK, and LOGJAM.

  • HTTP Security Header
    Apart from the SSL Certificate, we also check the HTTP Security Header. If configured correctly, HTTP headers such as Content Security Policy increase a page's security, protecting against cross-site scripting attacks.

  • DNS
    We check whether your site is using DNS security features like DNSSEC, CAA records, or DANE.
