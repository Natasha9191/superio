WordPress Pagespeed : Que faire si le site WordPress est trop lent ?

Auteur
Janis von Bleichert
Dernière mise à jour
26. janv. 2024

Un site WordPress qui se charge trop lentement incite les utilisateurs impatients à le quitter. Même un bon contenu ne convaincra pas les visiteurs de rester sur votre site. Des temps de chargement courts et une vitesse de page optimale (ou « Pagespeed ») sont donc essentiels pour le succès en ligne de votre site WordPress. Heureusement, il existe quelques mesures qui vous permettent d'atteindre cet objectif.

Dans cet article, nous vous expliquons comment optimiser la vitesse de page de votre site WordPress et ce à quoi vous devez faire attention.

Pagespeed : Qu'est-ce que c'est et pourquoi est-ce important ?

Le terme Pagespeed (en français : « vitesse de page ») décrit le temps de chargement d'un site web. Cela comprend le chargement de tous les contenus tels que les textes, les images et les scripts.

Il existe différents indicateurs permettant de mesurer ce temps de chargement, tels que le Largest Contentful Paint (LCP), qui est particulièrement apprécié. Il indique combien de temps il faut pour charger le plus grand élément de contenu d'une page et tient compte des images, de Javascript et de CSS.

Plus le LCP est bon – et plus le Pagespeed est rapide – plus vite une page est chargée et entièrement disponible pour les visiteurs. Un Pagespeed rapide est donc important pour de nombreuses raisons :

  • Une expérience utilisateur positive
    Des temps de chargement rapides améliorent sensiblement l'expérience utilisateur et sont aujourd'hui attendus par les visiteurs disposant d'une connexion Internet rapide. Les pages lentes frustrent généralement les visiteurs et les incitent à quitter le site.

  • Meilleur classement dans les moteurs de recherche
    Les moteurs de recherche comme Google s'appuient également sur des facteurs de performance comme le LCP pour classer les sites web. Des sites web plus rapides sont un atout, ce qui se traduit par une meilleure visibilité et davantage de visiteurs.

  • Taux de conversion plus élevés
    Les sites web avec des temps de chargement plus rapides ont souvent des taux de conversion plus élevés. Selon une étude de Portent, les taux de conversion sont trois fois plus élevés sur les sites dont le temps de chargement est d'une seconde que sur les sites qui mettent cinq secondes à se charger.

  • Une meilleure expérience sur smartphone
    Aujourd'hui, une grande partie du trafic de nombreux sites web est due aux utilisateurs de smartphones. Sur smartphone, les visiteurs sont encore plus impatients. De plus, la connexion Internet mobile peut être comparativement plus lente. Des temps de chargement optimisés sont donc encore plus importants pour les utilisateurs mobiles.

Comment mesurer le Pagespeed de mon site WordPress ?

Un bon Pagespeed est donc indispensable au succès de votre site WordPress. Mais avant de commencer à optimiser votre site, il est important de comprendre son état actuel.

La vitesse de chargement du site web sur vos propres appareils ne doit pas être considérée comme un indicateur. En effet, le navigateur met en cache de nombreux éléments des sites web qu'il consulte le plus souvent. Il s'agit donc avant tout des temps de chargement que les utilisateurs constatent lors de la première ouverture de votre site web.

Heureusement, il existe des outils spéciaux pour mesurer la vitesse de page des sites WordPress. L'un d'entre eux est le test PageSpeed d'EXPERTE.com. Vous pouvez y saisir l'URL du site web dont vous souhaitez mesurer la vitesse de page afin d'obtenir une liste de mesures pertinentes et un score de vitesse de page pour le site web et ses sous-pages :

Le test Pagespeed d'EXPERTE.com vous permet de mesurer en quelques secondes la vitesse de chargement de votre site web.

En moyenne, le temps de chargement des sites WordPress ne devrait pas dépasser 2 secondes. En outre, plus c'est rapide, mieux c'est – et quelques millisecondes peuvent tout changer. Si votre Pagespeed est mauvais, il est temps d'y remédier.

10 méthodes pour optimiser le Pagespeed de votre site WordPress

Il existe de nombreuses méthodes pour optimiser le Pagespeed. Voici dix méthodes auxquelles vous pouvez vous atteler immédiatement :

1.

Choisir un hébergement WordPress performant

Il est essentiel de choisir un bon service d'hébergement pour que votre site web soit performant. Un hébergement rapide et fiable garantit des temps de chargement plus courts et un site web généralement plus stable. Pour les projets WordPress, l'hébergement géré de WordPress, qui est entièrement adapté aux exigences de la plateforme WordPress, est une solution idéale.

2.

Utiliser les plugins de mise en cache

Les sites WordPress dynamiques se restaurent à chaque visite. Lorsque les utilisateurs se rendent sur votre site, ils doivent accéder aux bases de données et aux scripts de votre serveur, ce qui allonge les temps de chargement. Les plugins de mise en cache raccourcissent ce processus : ils « restaurent » votre site web à l'avance et fournissent à chaque utilisateur une vue pré-restaurée à partir de la mémoire cache.

Les navigateurs web tels que Chrome enregistrent les sites web localement sur votre ordinateur, de sorte qu'ils se chargent plus rapidement lors de visites répétées. Les plugins de mise en cache de WordPress fonctionnent de la même manière, sauf que le cache est enregistré sur le serveur de votre site web.

Dans la section des plugins de WordPress, vous trouverez de nombreux plugins de mise en cache.

Il suffit d'installer un plugin de mise en cache correspondant sur WordPress. Le choix est vaste – en voici quelques-uns :

  • W3 Total Cache (gratuit)

  • WP Super Cache (gratuit)

  • WP Rocket (payant)

  • LiteSpeed Cache (gratuit)

3.

Optimiser les images

Les images ont un impact important sur la vitesse de page. Plus il y a d'images sur une page, et plus les fichiers image sont volumineux, plus le site web se charge lentement. En même temps, les images sont assez importantes pour la plupart des sites web. En outre, il ne faut pas trop réduire la taille des images, sinon la résolution pourrait en souffrir.

Il existe quelques étapes pratiques pour optimiser efficacement la qualité des images et améliorer la vitesse de page de votre site WordPress :

  • Enregistrez les images pour le web : Les logiciels de traitement d'images tels que Gimp ou Photoshop offrent la possibilité d'enregistrer les images de manière ciblée pour une utilisation sur le web, afin de trouver le meilleur équilibre entre qualité et taille de fichier.

  • Sélectionnez le bon format de fichier : Le choix du format de fichier est également important. Les fichiers JPEG, par exemple, nécessitent beaucoup moins de mémoire que les fichiers PNG et offrent un bon équilibre entre la qualité et la taille du fichier. Une alternative intéressante est le format d'image WebP de Google, qui promet une compression 30 % meilleure que le JPG.

  • Utilisez des outils et des plugins : Vous pouvez utiliser des plugins comme WP-Smushit, PNGGauntlet, kraken.io ou tinypng pour automatiser la compression. Vous pouvez ainsi réduire la taille des fichiers image sans perte de qualité significative et améliorer ainsi le temps de chargement de votre site.

4.

Effectuer régulièrement des mises à jour

Les temps de chargement optimisés sont également importants pour les développeurs de la plateforme WordPress. C'est pourquoi ils améliorent régulièrement le code, ce qui peut avoir une influence positive sur la vitesse de chargement de votre site. Vous devriez donc toujours veiller à ce que votre installation WordPress soit à jour (ce qui est également important pour des raisons de sécurité).

Si vous n'avez pas désactivé cette fonction, WordPress effectue automatiquement les mises à jour de son logiciel. Dans le tableau de bord de WordPress, vous pouvez voir la version actuelle sous Tableau de bord > Mises à jour. Vous y trouverez également une liste des plugins et des thèmes pour lesquels des mises à jour sont disponibles. Ceux-ci devraient également être mis à jour afin d'optimiser la vitesse de page.

WordPress effectue régulièrement des mises à jour automatiques.

5.

Supprimer les plugins superflus

Sans les plugins, WordPress ne serait pas WordPress, mais il ne faut pas en abuser. Les plugins consomment des ressources, et en installer trop peut avoir un effet négatif sur la vitesse de page.

Réfléchissez donc pour chaque plugin s'il apporte vraiment une valeur ajoutée à votre site et vérifiez régulièrement s'il y a des plugins superflus que vous pouvez supprimer.

6.

Améliorer les temps de chargement avec des plugins de vitesse

Les plugins peuvent donc ralentir votre site web – mais il y en a aussi qui font exactement le contraire. Les plugins de vitesse peuvent aider à accélérer votre site web en réduisant la taille des fichiers CSS, HTML et JavaScript. Cette réduction est également appelée « minifying ».

Il existe de nombreux plugins qui permettent d'automatiser les processus de minifying et d'optimiser la vitesse de page de votre site WordPress. En voici quelques-uns :

  • Autoptimize : Ce plugin agrège et minimise HTML, CSS et JavaScript, déplace les scripts dans le pied de page et minimise HTML. Il peut également optimiser Inline JavaScript et CSS et offre des options pour optimiser les Google Fonts et les images.

  • WP Rocket : Nous avons déjà mentionné WP Rocket plus haut. Ce plugin de vitesse ne propose pas seulement des fonctions de mise en cache, mais optimise également votre site web grâce au minifying, au lazy loading pour les images, à l'optimisation de la base de données et à de nombreuses autres fonctionnalités.

  • WP-Optimize : WP-Optimize est également un outil de performance tout-en-un qui combine la mise en cache, le minifying et la compression d'images.

7.

Choisir un thème WordPress efficace

Le design de votre site web est la première chose que vos visiteurs remarquent. Le choix du thème WordPress, qui constitue la structure visuelle de votre site, est donc très important. Lors de votre choix, gardez néanmoins à l'esprit qu'un thème visuellement attrayant peut avoir un impact négatif sur le temps de chargement de votre site s'il n'est pas programmé efficacement.

Ce à quoi vous devez faire attention, en plus de la qualité visuelle, lors du choix du thème :

  • Programmation efficace : Évitez les thèmes surchargés de fonctions inutiles et de ce que l'on appelle les « bloatwares », car ils peuvent allonger le temps de chargement de votre site.

  • Tests de performance : Avant de choisir un thème, testez ses performances. Vérifiez la vitesse de chargement du thème et s'il a une influence sur le temps de chargement de votre site web. Le test Pagespeed d'EXPERTE.com peut être utile à cet égard.

  • Thèmes légers : Recherchez des thèmes optimisés pour la vitesse. Ces thèmes ont souvent un code plus léger et se chargent plus rapidement, ce qui améliore l'expérience utilisateur et optimise en même temps la vitesse de page de votre site web.

8.

Utiliser un réseau de diffusion de contenu (CDN)

Votre site web doit se charger rapidement pour tous les visiteurs, qu'ils y accèdent depuis Berlin ou Tokyo. Si votre site s'adresse à un public international, vous devriez envisager d'intégrer un réseau de diffusion de contenu (ou CDN, Content Delivery Network).

Un CDN stocke vos fichiers statiques, tels que CSS et JavaScript, sur des serveurs situés aux quatre coins du monde. Lorsque les visiteurs consultent votre site WordPress, ces fichiers sont chargés à partir de l'emplacement du serveur le plus proche géographiquement. Pour les utilisateurs éloignés de votre serveur principal, les temps de chargement peuvent ainsi être considérablement réduits.

De nombreux fournisseurs d'hébergement web disposent déjà d'un CDN intégré. Si ce n'est pas le cas chez votre hébergeur, vous pouvez recourir à des services externes tels que Cloudflare, bunny.net ou keycdn, qui s'intègrent en général facilement à WordPress.

9.

Optimiser les bases de données WordPress

Textes, images, podcasts, PDF – toutes les données qui composent votre site web sont stockées dans des bases de données. Et ces bases de données ne sont souvent pas structurées de manière très efficace. C'est surtout le cas pour les sites web qui sont en ligne depuis très longtemps et qui ont beaucoup de contenu.

Les anciennes bases de données, dont la structure est inefficace, peuvent entraîner des retards dans les temps de chargement. C'est là que vous pouvez intervenir pour optimiser le Pagespeed.

Il n'est pas recommandé de nettoyer soi-même les bases de données – le risque est trop grand que vous supprimiez irrémédiablement des données importantes. Vous pouvez plutôt utiliser un plugin qui se charge d'optimiser les bases de données WordPress à votre place, comme par exemple WP-Optimize ou WP-DB-Manager.

10.

Optimiser les colonnes de commentaires de votre site web

Votre site WordPress a-t-il une communauté active et des utilisateurs qui participent aux discussions dans les colonnes de commentaires ? Cela peut aussi avoir un impact sur les performances : les pages contenant des dizaines, parfois des centaines de commentaires, parfois même des fichiers image, peuvent mettre beaucoup de temps à se charger.

Vous pouvez facilement réduire les temps de chargement en répartissant les commentaires sur plusieurs pages. Pour cela, il suffit de naviguer dans le tableau de bord WordPress vers Paramètres > Discussion et de cocher la case « Répartir les commentaires sur les pages ». Vous pouvez définir le nombre de commentaires affichés par page et si les commentaires les plus anciens ou les plus récents doivent être affichés en haut.

Diviser les colonnes de commentaires en plusieurs pages pour améliorer la vitesse de page.

Conclusion

Vous êtes confronté à des temps de chargement longs sur votre site WordPress ? Ne paniquez pas ! Il existe de nombreuses mesures que vous pouvez prendre immédiatement pour améliorer sensiblement les performances de votre site. Des plugins de mise en cache à la compression des images en passant par le choix d'un hébergement rapide et fiable, chacune de ces mesures contribue à une amélioration des temps de chargement.

Mais tout d'abord, il faut vérifier ce qu'il en est des temps de chargement. Le test Pagespeed d'EXPERTE.com vous y aidera. Vous trouverez d'autres conseils sur la manière d'optimiser la vitesse de chargement de votre site Web (pas seulement sur WordPress) en cliquant ici.
