MX Lookup : Vérifier le serveur de messagerie

Avec le MX Lookup d'EXPERTE.com, déterminez rapidement et facilement les enregistrements du serveur de messagerie d'un domaine. Pour cela, il suffit de saisir le domaine souhaité dans le champ ci-dessus.

En réponse, vous obtenez un tableau avec les enregistrements MX du domaine.

Enregistrement MX - Structure

L'enregistrement MX indique sous quel (sous-)domaine le serveur de messagerie d'un (sous-)domaine est accessible. Cela est nécessaire pour que les e-mails envoyés à des adresses telles que exemple@domaine.com soient transmis au bon serveur.

L'enregistrement MX est structuré comme suit :

Teil

Description

Type

MX

Priorité

Indique la priorité de l'enregistrement (les enregistrements avec des valeurs inférieures sont privilégiés)

Serveur de messagerie

Nom du serveur de messagerie

TTL

Time to live (combien de temps l'enregistrement doit rester en cache)

MX Lookup - Déroulement

Pour envoyer un e-mail à info@experte.com, il faut suivre les étapes suivantes :

  • 1.

    L'expéditeur récupère les enregistrements MX d'experte.com et obtient :

1 aspmx.l.google.com.
5 alt1.aspmx.l.google.com.

...

2. L'expéditeur détermine maintenant l'enregistrement A pour le serveur de messagerie avec la plus haute priorité (aspmx.l.google.com) et obtient l'adresse IP suivante : 108.177.15.27

3. L'expéditeur envoie alors l'e-mail à l'adresse IP déterminée. Si cela échoue, l'expéditeur essaie le deuxième serveur de messagerie.
