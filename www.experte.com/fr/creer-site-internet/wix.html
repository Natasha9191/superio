Avis Wix : Toujours le meilleur ?

Auteur
Martin Gschwentner
Dernière mise à jour
15. févr. 2023

De la même manière que les programmes antivirus ont tendance à donner à leurs logiciels des noms commençant par la lettre « A », les logiciels de création de sites web semblent privilégier la lettre « W ». Weebly, Webnode, Webflow, mais aussi Wix, que nous allons découvrir dans cet article, font partie de ces logiciels. Cela signifie-t-il que la lettre « W » est un symbole de réussite ?

Est-ce que Wix fait partie des meilleurs logiciels de création de sites web ? Nous allons le découvrir à travers notre évaluation.

Qu'est-ce que Wix ?

Wix est une plateforme de développement web basée sur le cloud du développeur de logiciels du même nom, dont le siège se trouve à Tel Aviv. Grâce à son logiciel de création de sites web, les utilisateurs de Wix peuvent créer des sites web HTML complexes avec la fonction « Drag-and-drop » (« glisser-déposer ») et sans connaissances en programmation.

L'éditeur fonctionne selon le principe WYSIWYG, autrement dit « What you see is what you get », ce qui littéralement veut dire « Ce que vous voyez est ce que vous obtenez ». Cela signifie que les modifications effectuées dans l'éditeur apparaissent exactement telles qu'elles le seront sur les écrans une fois le travail terminé. À travers notre évaluation disponible sur experte.com/fr, nous avons examiné de plus près comment fonctionne Wix et s'il vaut le détour.

Installation et facilité d'utilisation

Avant de pouvoir utiliser le logiciel de création de sites web de Wix, vous aurez besoin de créer un compte. Cliquez sur le bouton bleu « Créer son propre site web » pour accéder à l'espace de connexion où vous pouvez vous inscrire avec votre adresse mail ou bien via vos comptes de réseaux sociaux.

La création d'un compte sur Wix se fait en un clin d'œil.

Une fois inscrit, vous serez redirigé vers l'espace de gestion de l'interface web de Wix. Celui-ci est structuré de manière claire : à gauche, vous trouverez la barre de menu qui vous permet de passer d'une section à une autre. Au centre vous trouverez différents widgets qui conduisent directement aux fonctions ainsi qu'aux sections individuelles comme votre blog ou vos factures. Un aspect pratique est que vous pouvez déterminer vous-même quels widgets doivent apparaître sur la page d'accueil de l'espace de gestion. Les notifications, les paramètres et le support sont accessibles via la barre de menu supérieure.

Si vous avez déjà créé un site web avec Wix, des données analytiques à ce sujet seront également mises à disposition dans l'espace de gestion.

L'espace de gestion de Wix, depuis lequel vous pouvez garder un œil sur vos projets web, est structuré de manière claire.

Vous pouvez accéder à l'éditeur de site web de Wix à proprement parler en cliquant sur « Nouveau site web ». Ici, vous devez d'abord indiquer la raison pour laquelle vous avez besoin d'un site web. Vous pouvez choisir parmi des catégories telles que « Entreprise », « Curriculum vitae & portfolio » ou « Blog ».

Pour quelle raison avez-vous besoin d'un site web ? Wix vous propose une sélection de templates en fonction de vos besoins.

Vous pouvez ensuite sélectionner l'éditeur que vous souhaitez utiliser pour créer votre site web. Pour les débutants, le logiciel propose l'outil d'automatisation ADI. Cette option basée sur l'IA (intelligence artificielle) permet de créer un site web personnalisé en quelques minutes, en fonction de vos besoins individuels.

Wix propose deux éditeurs différents. L'éditeur ADI basé sur l'IA est le plus simple des deux.

Éditeur ADI : création de sites web pour débutants

Avant que le moteur ADI ne se mette au travail, vous devez répondre à quelques questions relatives à votre entreprise et vos besoins. Vous pouvez également indiquer vos préférences en matière de design. Préférez-vous un design amusant ou plutôt sérieux ? Sur cette base, l'éditeur ADI vous propose différents templates.

Les templates de design sont moins variés dans l'éditeur ADI que dans l'éditeur Wix standard.

Pages prédéfinies

L'éditeur ADI rend les choses presque trop faciles : à l'étape suivante, vous pouvez sélectionner différentes pages, comme par exemple « À propos de nous », « Assistance » ou bien « Politiques », que vous pouvez ajouter à votre site web. Cliquez ensuite sur « Modifier le site web » afin d'accéder à l'éditeur ADI à proprement parler.

L'éditeur ADI est facile à utiliser, mais il lui manque de nombreuses fonctions.

Sur des pistes fixes

Contrairement à Jimdo, par exemple, qui comme Dolphin propose un service similaire, il est possible de passer ultérieurement de l'éditeur ADI à l'éditeur standard. Vous n'avez donc pas à craindre de perdre des fonctionnalités importantes à long terme.

Il manque aussi quelques fonctionnalités dans la version ADI. Vous pouvez également modifier votre site web dans l'éditeur ADI simplifié. Vous serez toutefois limité à des sections prédéfinies que vous pouvez organiser et modifier comme bon vous semble, au lieu de bénéficier d'une liberté totale avec la fonction « Drag-and-drop ». L'éditeur ADI facilite ainsi la tâche des créateurs de sites web qui n'ont pas l'œil pour le design, car les sections, disposées comme des blocs de Lego, sont toujours parfaitement coordonnées.

En cliquant sur le bouton « plus », vous pouvez ajouter de nouvelles pages et sections. Dans chaque section, vous pouvez modifier les contenus tels que les textes, les images ou bien les boutons, et adapter le design. Cependant, vous ne pouvez utiliser que certains templates de Wix. Vous ne remplissez donc vraiment que les espaces vides des templates prédéfinis avec vos contenus — la liberté de création n'existe pas dans l'éditeur ADI.

L'éditeur ADI est facile à utiliser mais il met un frein à votre créativité.

Dans l'ensemble, l'éditeur ADI marque des points grâce à son utilisation simple, mais en perd en termes de liberté de création. Les templates de design disponibles sont également très limités par rapport à ceux proposés dans l'éditeur standard. De plus, de nombreuses fonctionnalités et applications manquent complètement. Le service basé sur l'IA convient donc surtout aux débutants qui souhaitent perdre le moins de temps possible sur le design.

Ceux qui ne souhaitent pas se limiter dans la création de sites web avec Wix devraient plutôt opter pour l'éditeur standard, étant donné qu'il ne pose pas problème de changer d'éditeur. En revanche, si vous souhaitez revenir à la version ADI par la suite, vous perdrez touts vos progrès réalisés dans l'éditeur.

Plus de fonctions avec l'éditeur standard

L'éditeur standard vous permet d'être plus libre. Vous devez premièrement sélectionner votre template préféré parmi les templates répartis en différentes catégories et sous-catégories. En cliquant sur « Modifier » vous accédez ensuite à l'éditeur de site web.

Le choix de templates est nettement plus large est varié dans l'éditeur standard.

Étant donné que cet éditeur est un peu plus complexe que l'éditeur ADI, il faut un certain temps avant d'avoir une vue d'ensemble de toutes les fonctions. Dans la barre de menu supérieure, vous pouvez passer d'une page à une autre dans le menu déroulant et effectuer des réglages généraux. Dans le menu à gauche, vous pouvez ajouter des éléments individuels tels que des pages, des blocs, des applis, des fichiers et plus. En cliquant sur les éléments individuels sur l'espace de travail au centre, c'est-à-dire l'aperçu de votre site web, vous pouvez les modifier davantage.

En règle générale, vous pouvez déplacer et adapter librement toutes les choses que vous voyez sur votre espace de travail. Seuls certains éléments, par exemple votre menu de navigation, peuvent être modifiés dans un espace supplémentaire.

L'éditeur standard de Wix est plus complexe, mais permet une personnalisation beaucoup plus importante.

Dans l'éditeur standard, vous pouvez agrandir, déplacer et adapter chaque élément comme vous le souhaitez, avec des repères d'aide comme dans Photoshop afin de simplifier la disposition. Dans la rubrique « Outils » de la barre de menu supérieure, vous pouvez définir vous-même les outils d'aide (par exemple règles ou lignes de quadrillage) qui doivent apparaître. Vous y trouverez également une barre d'édition pratique que vous pouvez déplacer librement sur l'écran.

La barre d'édition plus longue vous aide à adapter les éléments du site web.

Même si on peut se sentir submergé au début, la logique qui se cache derrière Wix s'assimile vite. De plus, une fois que l'on quitte l'éditeur ADI quelque peu limité, on ne peut plus se passer des possibilités d'adaptation proposées dans l'éditeur standard. En revanche, le risque est plus élevé que les blocs et les éléments dépassent les alignements et que l'on doive travailler un peu plus longtemps pour obtenir un design cohérent.

L'éditeur standard, en particulier, consomme beaucoup d'énergie, ce qui peut entraîner des problèmes de performance sur certains PC. Dans l'ensemble, les deux éditeurs conviennent toutefois parfaitement à leurs objectifs légèrement différents et rendent la création de sites web accessible aux utilisateurs de tous niveaux.

Fonctionnalités

Nous avons déjà décrit plus en détail les deux éditeurs et leur fonction « Drag-and-drop »/de création de site dans la partie « Installation ». Mais même le meilleur éditeur se mesure en fonction de ce que l'on peut en faire et Wix offre suffisamment de liberté de création.

Les éléments que vous pouvez intégrer dans votre site sont si nombreux que la barre de menu paraît d'abord assez surchargée. La disposition logique permet néanmoins de trouver rapidement les fonctions recherchées.

Une grande sélection d'éléments

En plus des éléments habituels comme les éléments de texte, les galeries d'images ou l'intégration des réseaux sociaux tels que Facebook ou Spotify, vous trouverez également des éléments très spécifiques. Vous pouvez par exemple intégrer les avis de vos clients directement sur votre site web, proposer un formulaire d'abonnement à votre newsletter par e-mail ou encore faire la promotion des offres spéciales actuelles dans votre boutique en ligne.

Un aspect pratique est que les éléments modifiés peuvent également être enregistrés dans la rubrique « Mes designs », ce qui vous permet de les réutiliser sous la même forme.

Il existe d'innombrables éléments et blocs de construction préfabriqués que vous pouvez insérer dans votre site web.

Galerie de médias solide avec photos de stock

Dans la partie « Médias » du logiciel de création de sites web, vous trouverez de nombreuses photos et illustrations de qualité professionnelle, avec lesquelles vous pouvez agrémenter votre site web. Elles sont réparties dans des catégories telles que « Animaux & nature », « Musique » ou encore « Monde des affaires ».

Pour ce qui est de la précision de la recherche, Wix ne peut bien entendu pas se mesurer aux fournisseurs de photos de stock. Toutefois, grâce à l'intégration simple de Shutterstock et Unsplash, vous pouvez rapidement rechercher des contenus à la fois gratuits et payants qui ne sont pas inclus dans le forfait Wix.

La galerie d'images de Wix n'a pas ce que vous recherchez ? Vous trouverez peut-être votre bonheur sur Shutterstock ou Unsplash.

Blogs avec toutes les bases

Les fonctions de blog de Wix sont solides, même si elles ne sont pas aussi sophistiquées que d'autres domaines du logiciel de création de sites web. Dans certains templates, un blog est déjà intégré, dans d'autres, vous devez d'abord l'ajouter vous-même. Vous pouvez attribuer des tags à vos articles, les classer par catégories et effectuer des réglages SEO élémentaires. Vous n'êtes pas obligé de publier les articles immédiatement, mais vous pouvez programmer leur publication à l'avance.

App Market pour applications supplémentaires

Si les applications et les fonctions déjà intégrées ne vous suffisent pas, vous pouvez continuer à vous aventurer sur l'App Market de Wix qui est très vaste. Vous y trouverez des centaines d'extensions, dont beaucoup sont gratuites, qui vous permettront de personnaliser davantage l'expérience sur votre site web. Tout est intégré de manière transparente dans l'interface de Wix, de sorte que vous n'avez pas besoin de quitter votre site lorsque vous recherchez des fonctionnalités supplémentaires.

Vous trouverez des applications de Wix lui-même et d'autres fournisseurs tiers sur l'App Market.

Les sites web multilingues ne posent pas de problème

Grâce à Wix, la création de sites web multilingues est relativement simple. Sous Paramètres > Langue & région, vous pouvez faire en sorte que votre site web soit disponible en plusieurs langues en cliquant sur le bouton « Ajouter la fonction multilingue ». Vous pouvez traduire vos contenus automatiquement ou manuellement. Un menu de langues est ajouté à votre site web, permettant aux visiteurs de sélectionner leur langue préférée.

Créez votre logo rapidement

Wix propose également quelques autres fonctions spéciales. Dans l'espace de gestion, sous « Marketing & SEO », vous trouverez par exemple un créateur de logo simple. Vous pouvez ainsi laisser l'intelligence artificielle créer votre logo à partir de quelques éléments prédéfinis et vous laisser surprendre par le résultat.

Le résultat ne peut bien évidemment pas être comparé à un design fait sur-mesure spécialement pour vous. Mais ceux qui souhaitent trouver rapidement un logo simple pour leur produit pourraient s'en contenter. Dans l'évaluation réalisée par experte.com/fr sur les meilleurs créateurs de logo, Wix occupe même la première place à ce niveau.

Vous devez d'abord répondre à quelques questions pour que Wix puisse vous créer un logo.

Ceux qui ont des compétences en programmation peuvent également contribuer eux-mêmes au code source, ce que ne permettent pas tous les logiciels de création de sites web. Si vous souhaitez également travailler sur la création de votre site web lorsque vous êtes en déplacement, vous disposez d'une application d'accompagnement relativement efficace. Elle vous permet de rédiger des articles de blog, de gérer les produits disponibles dans votre boutique en ligne ou de discuter avec vos clients à travers la fonction de chat. Mais contrairement à l'application de Weebly par exemple, Wix ne propose pas la fonction « Drag-and-drop » dans son application mobile.

L'application pour smartphone de Wix vous permet de rédiger des articles de blog et de garder une vue d'ensemble sur vos projets de site web.

Il faudrait plusieurs pages pour décrire toutes les fonctionnalités de Wix. Le logiciel de création de sites web fait les choses bien plutôt que de les faire à moitié et parvient à convaincre sur toute la ligne grâce à un ensemble de fonctions très complet.

Templates & design

Wix ne manque pas non plus de templates de design. Vous pouvez choisir parmi plus de 800 templates différents, répartis par catégories de sites web — du moins si vous créez votre site web avec l'éditeur standard. Si vous vous fiez à l'éditeur ADI, vous serez beaucoup plus limité dans la sélection de templates de design.

Les templates sont relativement variés, mais la tendance qui se dessine est celle de sites web faciles à faire défiler et contenant beaucoup d'images. Parmi les nombreux templates, il devrait toutefois y en avoir pour tous les besoins.

Avec plus de 800 templates, la sélection de templates est particulièrement variée.

Pas de possibilité de changement de template avec l'éditeur standard

Dans un premier temps, la richesse des templates nous a presque échappé, car contrairement à WordPress par exemple, Wix ne permet pas de passer d'un template à un autre avec l'éditeur standard (alors que cela est possible avec l'éditeur ADI). Si vous souhaitez essayer un nouveau template de design, vous serez obligé de créer un nouveau site web.

Une fois le template sélectionné, vous pouvez modifier le schéma de couleurs, les arrière-plans et les polices de caractères sur votre site web dans la gestion du design. Vous pouvez ajouter des effets de survol et d'arrière-plan tels que le parallaxe directement dans les menus des éléments de l'écran.

Vous pouvez adapter les couleurs de votre site web et d'autres aspects du design comme bon vous semble.

Grâce à ses nombreux templates et ses options de design, Wix fournit une base plus que solide, même s'il nous manque la possibilité de changer de template, comme nous le connaissons avec WordPress et que, parmi les templates, beaucoup se ressemblent.

Commerce en ligne, marketing & référencement

La création de boutiques en ligne constitue l'un des piliers de Wix. Cela se reflète dans les vastes possibilités de commerce en ligne. Si votre template n'en contient pas déjà une, alors vous devez d'abord ajouter une boutique en ligne dans le menu d'éléments de votre site web. Cela se fait via l'application Wix Stores que vous trouverez également dans l'App Market.

Une boutique en ligne complète

Vous pouvez accéder au menu de gestion de votre boutique en ligne dans l'éditeur, sous Mon entreprise > Wix Stores. Vous y trouverez toutes les fonctions importantes de la boutique, à commencer par la gestion des produits. Vous pouvez y ajouter des textes de description individuels, différentes options de produits, le stock ainsi que d'éventuelles remises. Il est également possible de regrouper les produits dans des collections.

Dans la gestion des produits, vous définissez les produits qui sont en vente dans votre boutique.

Le logiciel met également à votre disposition différents outils de commercialisation. Vous pouvez créer des bons d'achat valables pour certains produits, services et collections. Dans les paramètres SEO, les pages de produits peuvent être optimisées pour les moteurs de recherche.

Sur votre site web, vous pouvez disposer les produits de votre répertoire comme vous le souhaitez et les intégrer comme d'habitude grâce à la fonction « Drag-and-drop ». Vous pouvez procéder à de nombreux réglages concernant la mise en page, le design et le tri de vos produits et collections. Différents widgets sont également disponibles.

Les widgets tels que les « galeries Slider » vous permettent de mettre en scène vos produits.

Une fois le design de votre site web terminé, vous n'avez bien sûr plus besoin de passer par l'éditeur pour gérer votre boutique. Vous trouverez également toutes les fonctions importantes, du portefeuille de produits aux commandes, dans l'espace de gestion de Wix.

Centre de marketing intuitif

Le tableau de bord général de Wix vous permet également d'avoir une vue d'ensemble de vos activités de marketing. La plupart des outils font partie de la plateforme Ascend, la suite professionnelle de Wix. Sur la page d'accueil marketing, vous pouvez créer et suivre des campagnes, par exemple de réseaux sociaux comme Facebook ou Instagram. Vous pouvez également y programmer et envoyer des newsletters par e-mail.

On y trouve également quelques autres fonctions supplémentaires. En plus du créateur de logo que nous avons déjà mentionné, Wix vous aide également à créer des vidéos de marketing. Grâce au créateur de vidéo, vous pouvez créer des vidéos en un clin d'œil et les partager sur les réseaux sociaux ou sur YouTube. Wix vous aide également à créer et à diffuser des annonces publicitaires sur Facebook et Instagram.

De manière générale, Wix vous prend — si vous le souhaitez — par la main avec sa plateforme de marketing Ascend et rend les campagnes intuitives même pour les débutants.

Qu'il s'agisse de réseaux sociaux ou de marketing par e-mail, Wix vous soutient dans la commercialisation de votre site web.

SEO & données analytiques

Wix met à votre disposition votre propre plan SEO pour optimiser l'ensemble de votre site web pour Google. Vous pouvez adapter vos contenus aux exigences des moteurs de recherche étape par étape sur la base du tutoriel et devenir vous-même expert du référencement de votre site web grâce à ce guide. Si vous vous y connaissez déjà, vous pouvez modifier vous-même les contenus pertinents pour le référencement, comme les méta-descriptions.

Dans le domaine des données analytiques, Wix propose l'intégration de Google Analytics, Yandex Metrica, Facebook Pixel et bien d'autres. Vous pouvez également ajouter des codes de suivi personnalisés pour suivre le nombre et le comportement des visiteurs sur votre site web et évaluer l'efficacité des campagnes.

Dans l'ensemble, Wix répond aussi à presque tous les besoins, y compris dans les domaines du commerce en ligne et du marketing.

Service d'assistance à la clientèle

Dans le centre d'assistance de Wix, de nombreuses instructions et FAQ sont disponibles, ce qui permet de dissiper de nombreux doutes dans les articles détaillés. Les articles sont judicieusement répartis en catégories et une fonction de recherche est bien entendu également disponible.

En revanche, Wix a quelque peu dissimulé la prise de contact directe — peut-être pour limiter le nombre de questions. Ce n'est que lorsque vous marquez un article comme « non utile » dans le centre d'assistance que le lien de contact apparaît.

Dans le centre d'assistance de Wix, vous trouverez de nombreux articles, mais les possibilités de contact pourraient occuper une place plus importante.

Ici, vous pouvez poser votre question et discuter avec un collaborateur disponible via le chat en direct. Nous n'avons pas eu à attendre longtemps : après quelques secondes, nous étions déjà en ligne avec un membre de l'équipe d'assistance qui s'est donné beaucoup de mal pour répondre à notre question.

Vous pouvez également demandé à être rappelé. Entre-temps, l'assistance téléphonique de Wix est disponible 24 heures sur 24.

Dans le chat d'assistance, nous avons été aidés au bout de quelques secondes seulement.

Nous avons trouvé l'assistance de Wix absolument rapide et fiable. De plus, les articles disponibles dans le centre d'assistance nous ont également aidés à comprendre l'un ou l'autre aspect de la plateforme. Il est toutefois dommage que Wix dissimule quelque peu le contact direct :

Tarification

En plus de la version gratuite qui lie les clients au domaine de Wix et qui manque en termes de fonctionnalités de base, Wix propose sept forfaits premium. Ceux-ci se subdivisent en quatre forfaits pour les sites web (Connexion Domaine, Basique, Illimité, VIP) et quatre forfaits pour les entreprises et le commerce en ligne (Business Basique, Business Illimité et Business VIP).

Tous les forfaits premium comprennent un domaine personnel, un certificat SSL gratuit et un service clientèle disponible 24 heures sur 24. La bande passante est limitée à 1 ou 2 Go dans les deux forfaits de site web les moins chers, sinon elle est illimitée. Pour ce qui est de l'espace de mémoire, les limites sont croissantes — de 500 Mo seulement pour le forfait Connexion Domaine jusqu'à 35 Go pour le forfait VIP et 50 Go pour le forfait Business VIP. Ce n'est qu'à partir du forfait Basique que Wix est sans publicité.

Cette renommée n'est pas un hasard et Wix n'est pas en tête de classement pour rien : le logiciel de création de sites web offre tout ce dont vous avez besoin et propose à la fois qualité et quantité. Les deux éditeurs sont simples à utiliser et intuitifs et l'App Market fait oublier les offres comparables des autres logiciels de création de sites web.

L'éditeur ADI a été un peu trop simplifié et pourrait permettre une personnalisation plus importante, mais comme pour d'autres logiciels de création de sites web avec service d'IA, nous recommandons de toute façon l'éditeur standard de Wix afin d'exploiter pleinement le potentiel des fonctionnalités. Seules les fonctions de blog pourraient être améliorées davantage. Dans l'ensemble, Wix est sans doute le choix le plus sûr en ce qui concerne les logiciels de création de sites web pour les utilisateurs de tous niveaux d'expérience.

Alternatives

Wix fait partie des logiciels de création de sites web WYSIWYG avec éditeurs « Drag-and-drop » les plus efficaces. Si vous souhaitez également travailler sur la création de votre site web en déplacement, cela ne sera pas possible avec l'application mobile de Wix. En revanche, cela est possible avec le logiciel de création de sites web Weebly, dont certaines des catégories font concurrence à Wix.

Si vous êtes à l'aise avec les langages de programmation et que vous cherchez un logiciel de création de sites web qui associe création visuelle et logique de codage, il vaut la peine de jeter un coup d'œil à Webflow.
