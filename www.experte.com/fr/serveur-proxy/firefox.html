Proxy Firefox : comment utiliser un proxy dans Firefox

Auteur
Janis von Bleichert
Dernière mise à jour
31. août 2022

Si vous souhaitez un peu plus d'anonymat sur Internet, les serveurs proxy valent le coup d'œil. Ils se placent entre vous et Internet en transmettant les demandes à votre place et en masquant votre adresse IP. La configuration d'un serveur proxy n'est pas si difficile et le navigateur Mozilla Firefox propose également différentes possibilités.

Il existe en principe trois méthodes pour établir une connexion proxy dans Firefox : vous pouvez vous rendre sur un simple service de proxy web, installer une extension de navigateur ou configurer manuellement le serveur proxy. Nous vous présentons brièvement ces trois méthodes.

1ère possibilité : proxy web

Il n'y a rien de plus simple que le proxy web : il suffit de se rendre sur le site web d'un proxy web et de saisir une URL dans le champ de recherche pour qu'une connexion proxy sécurisée soit établie avec le site web de votre choix.

L'avantage est que vous n'avez rien à installer ou à configurer. Selon l'outil proxy, il est tout au plus possible de procéder à quelques réglages simples, par exemple pour la gestion des scripts ou des cookies.

Le petit hic, c'est qu'il ne s'agit que d'une simple connexion HTTP directement établie vers un seul site web sélectionné manuellement dans le navigateur. Si vous souhaitez plus de flexibilité et que vous voulez par exemple faire passer le trafic d'une application de bureau par la connexion proxy, vous devez opter pour une autre méthode.

Les serveurs proxy web sont toutefois utiles pour les simples connexions proxy HTTP. De plus, il existe de nombreuses offres gratuites, par exemple de HideMyAss ou Hide.me :

Le fournisseur VPN HideMyAss propose également un service de proxy web gratuit.

Avantages et inconvénients

  • simple

  • gratuit

  • pas de configuration ou d'installation manuelle nécessaire

  • uniquement pour les connexions HTTP simples

  • chaque site web doit être visité manuellement via la page du proxy web

Vous êtes à la recherche d'un proxy web gratuit ? Dans notre article relatif aux serveurs proxy web, nous vous présentons sept fournisseurs.

2ème possibilité : extension de navigateur

Vous souhaitez vous détacher du proxy web et surfer de manière un peu plus flexible avec une connexion proxy ? Dans ce cas, une extension de navigateur proxy pourrait vous être utile. Elle vous permet d'appliquer la connexion proxy HTTP à l'ensemble du navigateur. De telles extensions de navigateur sont proposées par certains services VPN, par exemple :

1.

Proxy CyberGhost

L'extension proxy de CyberGhost propose quatre lieux : l'Allemagne, la Roumanie, les États-Unis et les Pays-Bas. Il n'y a malheureusement pas de limites de données, mais les paramètres avancés ne sont pas non plus possibles. Pour cela, vous avez besoin de l'application VPN premium du fournisseur, qui est promue dans l'extension de proxy. Dans l'évaluation réalisée par EXPERTE.com, cette application a tout de même obtenu la note globale de « bien ».

Avec l'extension proxy gratuite de CyberGhost, vous pouvez surfer sans limites de données.

2.

Proxy hide.me

Avec l'extension proxy de Hide.me, vous pouvez choisir entre des serveurs en Allemagne, au Canada et aux Pays-Bas. La sélection automatique du serveur est également entre vos mains. Il n'y a pas de limites de données et dans les paramètres, vous pouvez par exemple désactiver WebRTC et indiquer que la connexion proxy doit toujours être établie automatiquement.

Hide.me complète également son offre VPN avec sa propre extension de proxy.

3.

Proxy Hotspot Shield

L'extension proxy Firefox gratuite de Hotspot Shield ne vous impose pas non plus de limites de bande passante, mais propose en contrepartie (en plus de la sélection automatique du site) uniquement des serveurs aux États-Unis. Dans les paramètres, vous trouverez quelques fonctions de sécurité utiles, par exemple pour bloquer le tracking, les cookies ou les logiciels malveillants.

Avantages et inconvénients

  • pratiquement intégré dans le navigateur

  • gratuit

  • uniquement pour les connexions HTTP simples

  • choix limité de serveurs

  • selon le fournisseur, il existe des limites de données ou de vitesse

3ème possibilité : paramètres proxy manuels

Si vous souhaitez utiliser la connexion proxy de manière plus flexible, vous pouvez également la configurer manuellement. Pour cela, il vous faut d'abord l'adresse IP d'un serveur proxy que vous pouvez utiliser. Il existe de nombreuses offres gratuites, mais elles sont souvent peu fiables. Grâce à la liste de serveurs proxy d'EXPERTE.com, vous trouverez des IP de serveurs proxy gratuits qui sont régulièrement contrôlés :

Il existe bien entendu aussi des serveurs proxy fiables que vous pouvez louer contre paiement.

Dès que vous avez une IP proxy, vous pouvez configurer manuellement le serveur proxy dans Firefox. Nous vous expliquons étape par étape comment procéder :

1. Cliquez sur le menu latéral en haut à droite et dans le menu sur Paramètres.

Ouvrez d'abord les paramètres.

2. Faites défiler la page jusqu'aux Paramètres de connexion et cliquez sur Paramètres.

Dans les paramètres de connexion, vous trouverez les options de proxy.

3. Choisissez ensuite l'option Configuration manuelle du proxy. Saisissez ici dans le champ correspondant l'IP du proxy et le port correspondant, que vous trouverez par exemple dans la liste des serveurs proxy d'EXPERTE.com. Veuillez vérifier de quel type de proxy il s'agit (HTTP/S ou SOCKS5) et copiez les informations dans le champ correspondant.

Saisissez les données du proxy et cliquez sur OK pour finaliser la configuration.

4. Cliquez ensuite sur OK. Le serveur proxy est désormais activé.

Voilà comment ça fonctionne pour Mozilla Firefox. Si vous souhaitez également utiliser un serveur proxy avec d'autres navigateurs ou au niveau du système d'exploitation, nous avons rassemblé toutes les possibilités dans notre guide consacré à la configuration du proxy.

Avantages et inconvénients

  • flexibilité maximale

  • paramètres de proxy à l'échelle du système

  • les IP proxy gratuites doivent souvent être changées manuellement en raison de leur manque de fiabilité

  • pas de cryptage

Conclusion

Si vous souhaitez établir une connexion proxy dans votre navigateur Firefox, vous disposez de trois possibilités : vous pouvez utiliser un proxy web pour accéder à des sites web individuels via la connexion proxy anonyme. Avec une extension de navigateur, vous pouvez éviter de passer par le site web proxy et activer ou désactiver la connexion proxy directement dans la barre des tâches du navigateur.

Vous pouvez également configurer la connexion proxy directement dans les paramètres de Firefox. Pour cela, vous avez simplement besoin d'une IP proxy (gratuite), que vous trouverez par exemple dans la liste des serveurs proxy d'EXPERTE.com. Cependant, les serveurs gratuits sont souvent peu fiables par rapport aux offres gratuites. Il se peut donc que vous deviez d'abord essayer quelques IP proxy.

Les VPN constituent une alternative plus fiable et plus sûre au serveur proxy, mais ils sont généralement payants. Ils vous permettent également de masquer votre adresse IP et de contourner par exemple les blocages de pays, mais votre connexion est en outre cryptée. Dans l'article correspondant, vous apprendrez quels services VPN sont particulièrement adaptés à Firefox.
