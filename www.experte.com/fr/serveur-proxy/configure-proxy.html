Configurer un proxy : paramètres du proxy sous Windows, macOS, Android et iOS

Auteur
Janis von Bleichert
Dernière mise à jour
31. août 2022

Que ce soit sur PC, un smartphone ou une tablette, un serveur proxy vous permet de masquer votre adresse IP afin de surfer de manière plus anonyme et d'accéder à des contenus bloqués. Pour profiter de tous les avantages d'un serveur proxy, vous devez d'abord le configurer manuellement dans le système d'exploitation. Nous vous expliquons comment cela fonctionne, étape par étape et avec des captures d'écran.

Tout d'abord, pour la configuration de la connexion proxy, il est important de savoir quel appareil et quel système d'exploitation vous utilisez. Toutes les méthodes ont en commun le fait que l'on vous demandera tôt ou tard l'adresse IP d'un serveur proxy. Vous pouvez soit l'acheter contre paiement, soit la trouver gratuitement sur Internet. Dans notre liste de serveurs proxy, vous trouverez une liste d'IP de serveurs proxy toujours mise à jour.

Nous vous expliquons ci-dessous comment établir une connexion proxy pour Windows, macOS, Android et iOS :

Windows : configurer un serveur proxy

Comment configurer un serveur proxy sous Windows 11 :

1. Ouvrez d'abord les paramètres Windows et cliquez sur Proxy dans le menu Réseau et Internet.

Vous trouverez l'option proxy dans les paramètres de Windows.

2. Cliquez ensuite sur Utiliser un serveur proxy sous Configuration manuelle du proxy.

Accédez à la configuration manuelle du proxy.

3. Déplacez le curseur sous Utiliser un serveur proxy sur Activé et saisissez dans le champ l'IP du proxy et le port correspondant de notre liste de serveurs proxy. Vous pouvez également exclure certaines adresses de la connexion proxy ou indiquer que le proxy ne doit pas être utilisé pour les adresses locales.

Saisissez maintenant l'IP du proxy.

À partir de maintenant, votre connexion devrait passer par le serveur proxy (nous vous dirons plus tard comment tester si cela fonctionne vraiment).

macOS : configurer un serveur proxy

Voici comment configurer une connexion proxy sur macOS :

1. Ouvrez d'abord les Préférences système via le menu Apple et cliquez sur Réseau.

2. Sélectionnez le service réseau dont il s'agit dans la liste (par exemple « WLAN » ou « Ethernet »).

3. Cliquez sur Autres options, puis sur Proxy.

4. Sélectionnez ensuite le type de protocole et saisissez l'IP et le port correspondant. Vous trouverez toutes ces informations dans notre liste de serveurs proxy.

5. Cliquez sur OK, puis sur Appliquer pour finaliser la configuration. La connexion passe à présent par le serveur proxy.

Android : configurer le serveur proxy

Sur Android, vous devez configurer le serveur proxy séparément pour chaque réseau. Voici comment procéder :

1. Dans les paramètres d'Android, appuyez sur Réseau & Internet, puis sur Internet.

Ouvrez d'abord les paramètres Android.

2. Appuyez maintenant sur le réseau pour lequel vous souhaitez configurer la connexion proxy afin d'ouvrir les détails du réseau. Appuyez sur le symbole en forme de crayon en haut à droite pour accéder aux paramètres.

Vous trouverez les paramètres du proxy dans les détails du réseau.

3. Ouvrez les Options avancées et le menu de sélection sous Proxy et appuyez sur Manuel.

Par défaut, aucune connexion proxy n'est configurée sur Android.

4. Saisissez à présent sous Host Name l'adresse IP et le port proxy dans le champ correspondant. Pour pouvoir appuyer sur Enregistrer, vous devez confirmer une nouvelle fois le mot de passe WLAN.

Ici, vous devez avoir les détails de votre serveur proxy sous la main.

5. La connexion proxy est à présent établie, mais uniquement pour le réseau sélectionné.

iPhone : configurer un serveur proxy

Comment configurer un serveur proxy sur iOS :

1. Cliquez sur WLAN dans les Paramètres.

2. Cliquez ici sur la connexion WLAN pour laquelle vous souhaitez configurer le serveur proxy.

3. Cliquez sur Configurer le proxy en bas de la page, sous Proxy HTTP.

4. Sélectionnez Manuel pour définir vous-même l'IP et le port du proxy. Insérez ici les informations de notre liste de serveurs proxy.

5. Revenez aux paramètres WLAN pour enregistrer la configuration.

Vérifier la configuration proxy

Les serveurs proxy sont des intermédiaires invisibles qui opèrent en arrière-plan. Après les avoir installés, vous vous demandez donc peut-être s'ils fonctionnent et s'ils rendent votre connexion anonyme de manière fiable. Avec le contrôle IP d'EXPERTE.com, vous pouvez vérifier si l'adresse IP affichée de votre connexion correspond à celle du serveur proxy. Si c'est le cas, cela signifie que le serveur proxy est correctement configuré.

Connexion proxy lente

Votre connexion proxy n'est pas assez rapide ou n'est pas fiable ? Si vous utilisez une IP proxy gratuite, cela se produit probablement plus souvent. Avec les serveurs proxy gratuits, vous partagez la bande passante avec de nombreux autres utilisateurs, ce qui les rend généralement lents.

Si la vitesse n'est pas suffisante, vous pouvez choisir d'autres serveurs proxy dans notre liste. Vous pouvez également louer des serveurs proxy payants auprès de fournisseurs tels que Smartproxy, qui sont plus rapides et plus fiables.

Alternative : VPN

Si vous êtes à la recherche d'une méthode fiable pour changer régulièrement d'adresse IP et que vous souhaitez naviguer sur le web de manière encore plus anonyme, un service VPN est une bonne alternative au serveur proxy. Non seulement il masque votre IP pour contourner les blocages, mais il crypte en plus votre connexion pour la protéger des yeux de tiers.

Vous n'avez pas non plus besoin de configurer manuellement les services VPN. Au lieu de cela, vous établissez la connexion au niveau du système d'exploitation, indépendamment des paramètres de protocole, dans des applications conviviales. Toutefois, les offres VPN sont généralement payantes, même si certains des meilleurs services sont disponibles pour quelques euros par mois dans le tarif le plus avantageux.

Conclusion

Que ce soit Windows, macOS, Android ou iOS : les serveurs proxy sont installés en un tour de main dans tous les systèmes d'exploitation. Pour cela, vous avez seulement besoin d'une IP proxy, que vous pouvez par exemple trouver dans notre liste de serveurs proxy. Pour savoir si la connexion proxy fonctionne vraiment, vous pouvez vérifier votre adresse IP avec notre contrôle IP.

Vous trouverez tout ce que vous devez encore savoir sur les serveurs proxy dans notre article de synthèse. Si vous recherchez une méthode encore plus sûre et plus fiable pour masquer les IP, que vous ne devez pas configurer dans le système d'exploitation, cela vaut la peine de jeter un coup d'œil aux VPN.
